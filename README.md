# CloudCart – Enterprise Cloud-Native E-Commerce Platform

## Overview

CloudCart is a cloud-native e-commerce platform built to demonstrate modern DevOps practices using open-source tools and cloud-native technologies.

The project is being developed step by step following a real-world software development workflow instead of a tutorial-based approach.

## Project Goals

- Build a production-inspired cloud-native application
- Containerize services using Docker
- Deploy applications on Kubernetes
- Automate infrastructure using Terraform
- Implement CI/CD with GitHub Actions
- Follow GitOps using ArgoCD
- Monitor applications using Prometheus and Grafana
- Centralize logs using Loki
- Apply DevSecOps best practices

## Tech Stack

- Docker
- Kubernetes
- Terraform
- GitHub Actions
- Helm
- ArgoCD
- Prometheus
- Grafana
- Loki
- Trivy
